    Great game for great mind !! Math Practice app is basically designed keeping in mind the target audience and provide the easiest to play this game and enhance user maths skills. Aims of this game is to improve your maths mental exercise and make your calculation fast in a funny way.

In this game you have to just match the make equally number that display on the top corner. For example if it displays 18 then you have to chose a number from square box and complete 18 even you can do the addition and multiplication to find the no. like (9+9) or (6*3).

    Feature:
  • Addition and Multiplication support.
  • Easy And Challenge Level
  • Game Leaderboard and Achivement
